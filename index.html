ECHO_ENGINE: Reduzindo o Barulho para Crianças Autistas

Desenvolvido por Sandra Konvex, o ECHO_ENGINE é um dispositivo que monitora sons em tempo real para criar ambientes escolares mais inclusivos. Conheça os três modelos:

Portátil

Compacto, ideal para uso individual ou em casa. Ver repositório

Básico

Projetado para salas de aula, promove autorregulação coletiva. Ver repositório

Rede Neural (Edge Impulse)

Identifica tipos de sons com IA e está em fase de finalização. Ver repositório

Todos os modelos enviam dados para a AWS para análise futura. Saiba mais no repositório principal ou entre em contato!
